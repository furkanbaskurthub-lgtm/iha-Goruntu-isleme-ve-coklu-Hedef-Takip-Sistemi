# 🚀 GitHub'a Yükleme Komutları

## ✅ Hazırlık Tamamlandı!

Tüm dosyalar GitHub için hazır. Şimdi sadece aşağıdaki komutları çalıştır:

---

## 📝 Adım Adım Komutlar

### 1. Git Durumunu Kontrol Et
```bash
git status
```

### 2. Değişiklikleri Stage'e Al
```bash
git add .
```

### 3. Commit Yap
```bash
git commit -m "Initial commit: İHA Görüntü İşleme ve Hedef Takip Sistemi

- YOLOv8 ile nesne tespiti
- ByteTrack ile çoklu hedef takibi
- Profesyonel İHA HUD overlay
- GPS ve telemetri özellikleri
- Docker desteği
- Demo video eklendi"
```

### 4. Remote Repository Ekle
```bash
git remote add origin https://github.com/furkanbaskurthub-lgtm/iha-Goruntu-isleme-ve-coklu-Hedef-Takip-Sistemi.git
```

### 5. Branch Adını Main Yap
```bash
git branch -M main
```

### 6. GitHub'a Push Et
```bash
git push -u origin main
```

---

## 🎯 Tek Komutta (Hepsi Birden)

```bash
git add . && git commit -m "Initial commit: İHA Görüntü İşleme ve Hedef Takip Sistemi" && git remote add origin https://github.com/furkanbaskurthub-lgtm/iha-Goruntu-isleme-ve-coklu-Hedef-Takip-Sistemi.git && git branch -M main && git push -u origin main
```

## ⚠️ Eğer Hata Alırsan

### "remote origin already exists"
```bash
git remote remove origin
git remote add origin https://github.com/furkanbaskurthub-lgtm/iha-Goruntu-isleme-ve-coklu-Hedef-Takip-Sistemi.git
```

### "failed to push"
```bash
git pull origin main --allow-unrelated-histories
git push -u origin main
```

### "Authentication failed"
GitHub'da Personal Access Token oluştur:
1. GitHub → Settings → Developer settings → Personal access tokens
2. "Generate new token (classic)"
3. Repo seç
4. Token'ı kopyala
5. Push ederken şifre yerine token'ı kullan
